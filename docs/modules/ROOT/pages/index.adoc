= About

The library `PPWCode.AspNetCore.Host.I` is part of the .NET PPWCode project and is meant to be used as the for the Host project in a REST API solution implemented with ASP.NET Core.

This library is supported on .NET SDK versions that are currently supported by Microsoft as part of their official support lifecycle.


== Development

Development of the PPWCode is done on https://github.com/peopleware/net-ppwcode-aspnetcore-host/tree/I/next[GitHub].

Releases (both stable and pre-release) are published on https://www.nuget.org/packages/PPWCode.AspNetCore.Host.I/[NuGet] by https://www.nuget.org/profiles/PeopleWare[PeopleWare].
