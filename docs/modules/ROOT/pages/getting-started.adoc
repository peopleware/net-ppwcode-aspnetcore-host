= Getting started

== Add NuGet package

This package is available on NuGet: https://www.nuget.org/packages/PPWCode.AspNetCore.Host.I[PPWCode.AspNetCore.Host.I].

You can use your IDE to add it as a dependency to your project, or manually add it in the `.csproj` file.  This would look similar to the following:

[source,xml]
----
include::example$project.csproj[]
----

After the package is added, you can use the helper classes from `PPWCode.AspNetCore.Host.I` in your code.
